# Dev Log AI

> 개발자를 위한 기술 블로그 글 자동 생성 도구  
> OpenAI API를 활용해 **주제 + 키워드 + 글 유형**만으로 실사용 가능한 Markdown 글을 생성하는 서비스

## 🚀 프로젝트 소개

**Dev Log AI**는  
“블로그 글 쓰는 데 드는 시간과 에너지를 줄이자”는 문제의식에서 시작한 개인 프로젝트다.

개발하면서 정리해야 할 내용은 많은데

- 구조 잡기 귀찮고
- 마크다운으로 정리하기 번거롭고
- SEO까지 신경 쓰려면 더 피곤한

문제를 해결하고 싶었다.

이 프로젝트는 AI를 글을 대신 써주는 도구가 아니라  
**글 작성을 가속하는 개발 도구**로 사용하는 데 초점을 둔다.

## 🎯 개발 목표

- OpenAI GPT API 실제 서비스 수준으로 활용해보기
- 프롬프트 설계를 통해 **출력 품질을 통제**하는 경험
- Markdown 기반 콘텐츠 생성 → 편집 → export 전체 플로우 구현
- “실제로 내가 쓸 수 있는 도구” 완성

---

## 🧠 핵심 아이디어

- 글의 품질은 **모델보다 프롬프트 구조**가 결정한다
- 글 유형(Tutorial / TIL / Trouble Shooting)에 따라  
  → **문단 구조와 작성 규칙을 고정**
- AI는 자유롭게 쓰게 두지 않고  
  → **미리 정의한 글 설계도 안에서만 작성**

---

## ✨ 주요 기능

### 1. AI 기반 글 자동 생성 (Core Feature)

다음 입력값을 기반으로 글을 생성한다.

- **주제**
- **키워드**
- **글 유형 (Template)**

글 유형에 따라:

- 문단 구조
- 소제목 규칙
- 코드 블럭 포함 여부
- 톤 & 분량  
  이 프롬프트 레벨에서 제어된다.

---

### 2. 글 템플릿 기반 프롬프트 설계

현재 정의된 글 유형:

- **Tutorial**
- **TIL (Today I Learned)**
- **Trouble Shooti**
